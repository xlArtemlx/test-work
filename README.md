# Getting Started

## Step 1: Clone project
git clone https://github.com/your-username/your-repo-name.git

cd your-repo-name

## Step 2: Install Npm packg.
npm install


## Step 3: IOS Setup
cd ios

bundle install

pod install

cd ..

## Step 4: Running the App

# using npm
npm run ios

# using npm
npm run android
